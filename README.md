# FuckDailyCP
#### 简介
今日校园自动打卡脚本。
可以挂服务器上定时自动打卡。
健康的人可以用一下（雾）。
![](doc/img.png)
#### 使用说明
理论上支持所有学校。目前已知两种学工号登录方式，均已实现。
运行以下命令即可
```bash
python3 DailyCP.py 学校全名 学号 密码 定位地址
```
#### 挂服务器提示
请注意系统时间是否设置正确

Linux下可以使用`corntab -e`设置定时任务让程序每天自动打卡
```bash
0 9 * * * python3 DailyCP.py 学校全名 学号 密码 定位地址
```
